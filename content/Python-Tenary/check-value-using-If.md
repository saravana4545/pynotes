```python
# Ternary Operators
```


```python
# syndax - value_if_true if condition else value_if_false.
```


```python
# Example 1: Check if a number is even or odd
```


```python
number = 10
```


```python
result = "Even" if number % 2 == 0 else "Odd"
```


```python
print(result)
```

    Even



```python
# Example 2: Find the maximum of two numbers
```


```python
a, b = 5, 8
```


```python
maximum = a if a > b else b
```


```python
print(maximum)
```

    8
