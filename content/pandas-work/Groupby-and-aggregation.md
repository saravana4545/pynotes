```python

```


```python
agg_df = df.groupby('Department').agg(
    Total_Salary=('Salary', 'sum'),
    Avg_Experience=('Experience', 'mean'),
    Max_Salary=('Salary', 'max')
).reset_index()
```


```python
print(f"Grouped and Aggregated DataFrame:\n{agg_df}")
```
