```python
def leap_year(year):
    if (year % 4 == 0 and year % 100 != 0):
        return f"{year} is a leap year."
    else:
        return f"{year} is not a leap year."
```


```python
#put a range for it and print
```


```python
for i in range (1996,2000):
    print(leap_year(i))
```
