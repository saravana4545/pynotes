```python
def convert_to_uppercase(s):
    """
    Converts a given string to uppercase.

    Parameters:
    s (str): The input string to be converted.

    Returns:
    str: The input string converted to uppercase.
    """
    converted_list = s.upper()
    return converted_list
```

# Sample test cases
