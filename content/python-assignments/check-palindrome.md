```python
def is_palindrome(s):
    """
    Problem: Write a function to check if a string is a palindrome.
    
    Args:
    s: input string to be checked
    
    Returns:
    True if the string is a palindrome, False otherwise
    """
    if s == s[::-1]:
        return True
    return False
```
