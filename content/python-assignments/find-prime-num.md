```python
#7. Write a function to check if a number is prime.
```


```python
def is_prime(num):
    """
    Check if a number is prime.

    Parameters:
    num (int): The number to check for primality.

    Returns:
    bool: True if the number is prime, False otherwise.
    """
    if num % 2:
        return True

    return False
```
