```python
#8. Write a function to sort a list in ascending order.
```


```python
def sort_list_ascending(lst):
    """
    Sort a list in ascending order.

    Parameters:
    lst (list): List to be sorted.

    Returns:
    list: Sorted list in ascending order.
    """
    sorted_list = sorted(lst)

    return sorted_list
```
