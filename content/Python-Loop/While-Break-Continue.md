```python
# Break Condition
```


```python
i = 1
```


```python
while i <= 5:
    if i == 3:
        break
    print(i)
    i += 1
```

    1
    2



```python

```


```python
# Continue
```


```python
i = 1
```


```python
while i <= 5:
    if i == 3:
        i += 1
        continue
    print(i)
    i += 1
```

    1
    2
    4
    5
