```python
# Bitwise Left Shift (<<)
```


```python
a = 5  # Binary: 0101
```


```python
result = a << 1  # Binary: 1010 (5 * 2)
```


```python
print(result)
```

    10

```python
# Bitwise Right Shift (>>)
```


```python
a = 5  # Binary: 0101
```


```python
result = a >> 1  # Binary: 0010 (5 // 2)
```


```python
print(result)
```

    2
